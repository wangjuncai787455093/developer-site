# 链接块

链接容器，功能同链接组件，不过可以嵌入其他内容，例如图片等。

## 何时使用

- 切换访问当前应用的其他页面；
- 跳转到外部页面；

## 组件示例

import Iframe from 'components/Iframe';

<Iframe url="https://www.aliwork.com/developer/link-bolck-v2?isRenderNav=false" />
